Liste des événements

info

Ceci est une liste non exhaustive des événements disponibles avec Minecraft Forge. Il vous est cependant possible de contribuer à cette liste. Pour cela vous pouvez proposer une modification de ce fichier.

Événements concernants les blocs​

BlockEvent.BlockToolModificationEvent

Cet événement se déclenche lorsque le joueur clique sur un bloc avec un outil en main pour changer l'état du bloc (utilisé par exemple pour déterminer si une hache peut retirer les écorces d'une bûche).

Annulable : ✔
BlockEvent.BreakEvent

Cet événement se déclenche quand un bloc est cassé.

Annulable : ✔
BlockEvent.CreateFluidSourceEvent

Cet événement se déclenche pour permettre de vérifier si un bloc qui n'est pas une source peut se transformée en cette dernière.

Annulable : ❌
BlockEvent.CropGrowEvent

Cet événement se déclenche quand une culture grandi.

Annulable : ❌
BlockEvent.EntityMultiPlaceEvent

Cet événement se déclenche quand le fait de poser un bloc provoque la création de multiples blocs (comme par exemple avec les lits).

Annulable : ✔
BlockEvent.EntityPlaceEvent

Cet événement se déclenche quand un bloc est placé.

Annulable : ✔
BlockEvent.FarmlandTrampleEvent

Cet événement se déclenche lorsqu'une terre labourée est piétinée (re-transformée en terre).

Annulable : ✔
BlockEvent.FluidPlaceBlockEvent

Cet événement se déclenche lorsqu'un liquide place un bloc (comme pour un générateur de pierres ou du feu allumé par de la lave à proximité).

Annulable : ✔
BlockEvent.NeighborNotifyEvent

Cet événement se déclenche quand une mise à jour physique se produit sur un bloc.

Annulable : ✔
BlockEvent.PortalSpawnEvent

Cet événement se déclenche quand un portail du Nether est sur le point d'apparaitre.

Annulable : ✔
