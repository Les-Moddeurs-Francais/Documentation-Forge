## Définir le type de rendu

Pour définir le type de rendu d'un bloc, il est juste nécessaire d'ajouter une ligne dans le(s) fichier(s) modèle de votre bloc.

| Nom                      | Identifiant                          |
|--------------------------|--------------------------------------|
| SOLID                    | `minecraft:solid`                    |
| CUTOUT_MIPPED            | `minecraft:cutout_mipped`            |
| CUTOUT                   | `minecraft:cutout`                   |
| TRANSLUCENT              | `minecraft:translucent`              |
| TRANSLUCENT_MOVING_BLOCK | `minecraft:translucent_moving_block` |
| TRANSLUCENT_NO_CRUMBLING | `minecraft:translucent_no_crumbling` |
